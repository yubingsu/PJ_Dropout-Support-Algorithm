[2024] 학업 중단 원인의 다면적 분석을 통한 지원 알고리즘 구축

## 프로젝트 개요
<img width="1022" height="600" alt="image" src="https://github.com/user-attachments/assets/3ed7773e-5a49-4cf4-bc85-cb4d9512abd5" />
- **“어떤 지역·환경의 학생이, 어떤 이유로 학업 중단 위험이 높은가?”** 를 데이터로 규명
- 기존 정책은 학업 중단을 단일 요인으로 접근하는 한계가 있어, **지역·학교급·개인 특성을 고려한 다면적 분석과 선제적 지원 체계 필요** 
  

## 핵심 내용
- **학업 중단 추이 분석** : 2013~2023년 중·고등학교 학업 중단율 변화 추이 분석 -> 중학교 대비 고등학교에서 학업 중단율 변동성이 크고, 지역 간 격차가 구조적으로 유지됨을 확인
- **학업 중단 사유 체계화 및 요인 분류** : 학업 중단 사유를 개인·가정·학교·지역사회 요인으로 재분류, 학교급(중·고)에 따라 주요 위험 요인이 상이함을 구조적으로 정리
- **행정구역 단위 심층 분석** : 서울시 25개 행정구역을 대상으로 연도별 학업 중단 변화 추이 비교 -> 학업 중단자가 지속적으로 많은 지역과 적은 지역 간 격차가 고착화됨을 확인, 강남구·노원구·강서구·송파구 등 **특이 패턴 지역 도출**
- **외생 변수 결합 분석** : 소득 수준, 복지 예산, 교원 1인당 학생 수 등 지역 환경 변수와 학업 중단율 간 관계 분석 -> 중학교·고등학교에서 영향을 미치는 외생 요인이 다르게 작용함을 확인
- **지원 알고리즘 관점의 해석** : 단순 예측이 아닌, **“어디에 어떤 유형의 지원이 우선적으로 필요한가”**에 초점을 둔 결과 해석

## 주요 성과
- 중·고등학교 학업 중단을 **단일 현상이 아닌 다층적 구조 문제로 재정의**
- 행정구역·학교급·사유 유형을 결합한 분석을 통해 **차별화된 학생 지원 전략 필요성 도출**
- 학업 중단 위험이 높은 지역·유형을 선별해 **선제적 개입이 가능한 데이터 기반 정책 인사이트 제시**
- 향후 **학생 맞춤형 지원 알고리즘**으로 확장 가능한 분석 프레임 구축

## 기술
* Python, Pandas, Numpy  
* Matplotlib, Seaborn  
* Statsmodels  
* 공공데이터 기반 패널 데이터 분석

## 데이터 출처
* 교육통계서비스(KESS) – 행정구역별 학업 중단률 및 중단 사유 (2013–2023)  
* 서울특별시 교육청 통계  
* 서울특별시 소득·복지·교육 환경 관련 공공 데이터

## 파일 구조

- `regression_edit_final.ipynb` : 학업 중단율에 영향을 미치는 요인 회귀 분석
